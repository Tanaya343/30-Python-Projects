# 📝 Text Summarizer – GUI App

A simple Python GUI application that summarizes long text passages by extracting the most important sentences based on word frequency.

## 🚀 Features

- Summarizes any paragraph or article into a shorter version
- GUI built with Tkinter for easy interaction
- Calculates reading time and word count for original vs summary
- Uses basic NLP techniques with NLTK

## 📌 How It Works

1. **Input**: User enters or pastes a paragraph into the app.
2. **Tokenization**: The text is split into sentences and words.
3. **Frequency Analysis**: Counts how often each word appears (excluding punctuation).
4. **Scoring**: Sentences are scored based on word frequencies.
5. **Summary**: Top 30% highest-scoring sentences are extracted and displayed.
6. **Stats**: Word count and estimated reading time for original and summary.

## 🛠️ Tech Stack

- Python 3
- Tkinter – for GUI
- NLTK – for natural language processing
- Collections – for frequency dictionary
